# ROADMAP

Планы по продукту: локальная AI-платформа корпоративных знаний (ingest → RAG → ответы с цитатами → отчёты/презентации), ориентир — рынок РФ (локальный контур, контроль утечек, аудит).

## V1 — локальный Q&A по документам

**Цель:** пилот в контуре заказчика с доказуемыми ответами.

**Скоуп:** разделы; загрузка `PDF` / `DOCX` / `PPTX` / `XLSX` / `TXT` / `HTML`; парсинг, chunking, embeddings, индекс; чат по разделам; обязательные citations; базовый RBAC (`admin` / `member`); минимальный audit log; allowlist моделей и no-egress по умолчанию; экспорт ответа в markdown/text.

**Не в V1:** SSO enterprise, видео/ASR, Marp/презентации в продукте, multi-tenant SaaS «для всех», агенты, HA/DR.

**Гейт:** ответы без цитат не допускаются (или явный «в корпусе не найдено»); удаление документа инвалидирует индекс; воспроизводимый деплой on-prem; human eval на 20–50 вопросах заказчика.

## V2 — артефакты из знаний

**Цель:** brief / report / Q&A pack / executive summary из шаблонов; презентации через Marp с привязкой тезисов к источникам; speaker notes с цитатами; ACL на разделы и группы; read-only шаринг внутри контура; job queue для тяжёлого парсинга и реиндекса; retention; опционально гибридный поиск BM25+vector при подтверждённой необходимости; eval harness (golden questions, регрессия качества).

**Не в V2:** полноценный ECM/DMS, e-discovery suite, immutable SIEM-audit уровня enterprise, замена DLP/GRC.

**Гейт:** факты в report без цитат не выводятся; Marp pipeline воспроизводим (артефакты версионируемы).

## Enterprise edition — закупка ИБ и эксплуатация

**Цель:** соответствие чеклистам ИБ, интеграция в контур идентичности и эксплуатация в production.

**Скоуп:** SSO (SAML/OIDC), маппинг групп в ACL; расширенный audit, экспорт в SIEM, retention по политике; изоляция тенантов/разделов; метки чувствительности и политики смешивания контекста; шифрование at-rest, KMS по возможности; HA, backup/restore, healthchecks; SLA и security patch cadence; controlled export (watermarking, ограничения выгрузки); approved knowledge snapshots.

**Гейт:** threat model и закрытие P0/P1; DR drill по восстановлению из бэкапа; письменные ответы на типовой security questionnaire.

## Видео (модуль после V1)

Upload → ASR в локальном контуре → transcript → chunking → RAG; ответы с таймкодами. Зависит от job queue и хранилища. Не смешивать с must-have V1 без отдельного продакт-решения.
