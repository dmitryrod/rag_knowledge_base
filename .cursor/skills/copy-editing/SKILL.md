# Copy Editing

## Preconditions

- `.cursor/marketing-context.md` — voice, words to avoid

## Checklist

- Убрать штампы («инновационный», «стримлайнить» без сути)
- Активный залог; короче предложения
- Одна мысль на абзац; убрать дубли
- CTA конкретный (что получит пользователь)
- Проверить согласованность обещания с контекстом продукта

## Output

- Отредактированный текст + **Track changes summary** (что и зачем)
