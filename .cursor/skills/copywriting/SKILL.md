# Copywriting

## Preconditions

- `.cursor/marketing-context.md` — особенно аудитория, доказательства, tone

## Core principles

- Clarity over cleverness; benefits over features; specificity over vague AI adjectives
- One primary CTA per page/section where applicable
- Line-edit pass → при необходимости отдельно вызвать skill `copy-editing`

## Framework appendix (A/B)

Используй **один** фреймворк явно в черновике:

| Framework | Когда |
|-----------|--------|
| PAS | Прямой отклик, боль острая |
| AIDA | История, эмоция, бренд |
| BAB | Трансформация до/после |
| 4 U's | Заголовки: Useful, Urgent, Unique, Ultra-specific |

Headline patterns: `How to [outcome]`, `[N] ways to [benefit]`, `[Who] + [benefit]`, `[Problem]? [Solution].`

## Output

- Секции с headline / subhead / CTA
- 2–3 альтернативы hero + rationale
- Meta title/description если нужно
