# Marketing Ideas

## Preconditions

- `.cursor/marketing-context.md` — стадия, аудитория, ресурсы

## Modes

### Full (library-style)

Категории: content/SEO, paid, PLG, partnerships, community, email, launch…  
Для каждой идеи: название, почему подходит, первые шаги, метрика успеха, ресурсы.

### Quick (phuryn-style merge)

Ровно **5 идей**:

1. Channel  
2. Core message  
3. Why it works  
4. Cost / resource efficiency
