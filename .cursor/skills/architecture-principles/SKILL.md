# Architecture Principles

Чеклист для reviewer-senior (Level 2 — Senior Review).

## Архитектура и дизайн

- [ ] Разделение ответственности (SRP)
- [ ] Зависимости направлены в одну сторону (не циклические)
- [ ] Абстракции оправданы (не over-engineering)

## Граничные случаи

- [ ] Обработка ошибок и исключений
- [ ] Пустые/null/undefined входы
- [ ] Предельные значения (пустые списки, большие объёмы)

## Производительность

- [ ] N+1 запросы, лишние циклы
- [ ] Кеширование при повторных тяжёлых операциях

## Поддерживаемость

- [ ] Техдолг и его приоритет
- [ ] Расширяемость без ломающих изменений

## Вывод

Архитектурная оценка, риски, рекомендации. Не дублировать reviewer-senior Level 1 (линтеры, стиль).
