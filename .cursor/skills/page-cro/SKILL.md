# Page CRO

## Preconditions

- `.cursor/marketing-context.md`  
- Знать тип страницы и **один** главный conversion goal

## Audit dimensions (порядок impact)

1. Value proposition clarity  
2. Headline / message match к источнику трафика  
3. CTA hierarchy  
4. Visual hierarchy  
5. Trust / proof  
6. Objections  
7. Friction (форма, скорость, mobile)

## Appendix: CRO process (merge)

1. Map funnel → найти high-traffic / low-conv  
2. Hypothesis: если X → Y потому что Z  
3. Prioritize **PIE** (Potential, Importance, Ease) 1–10 each  
4. Test: sample size, неделя полных циклов, mobile отдельно

## Output

Quick wins / High impact / Test ideas / Copy alternatives
