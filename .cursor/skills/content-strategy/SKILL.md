# Content Strategy

## Preconditions

- `.cursor/marketing-context.md`

## Searchable vs shareable

| Type | Когда |
|------|--------|
| Searchable | Есть реальный поисковый спрос; закрыть интент полностью |
| Shareable | Инсайт, данные, лидерство мнений — без обязательного volume |

## Lenny appendix (merge)

- Проверить, что люди **ищут** тему; иначе контент не «болеутолкающий»  
- Content-market fit: тревоги профессии, не generic advice  
- Evergreen: бюджет времени на обновление старых постов  
- AI: assist, not replace — добавлять information gain человека

## Outputs

- 3–5 pillars + topic cluster map  
- Приоритетная таблица тем (impact, fit, search, effort)
